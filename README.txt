Prompt Generator Tool - README

This tool allows you to create structured, AI-ready prompts with ease. To use:

1. Open `index.html` in your web browser.
2. Fill in the required fields and select desired options.
3. Click 'Generate Prompt' to see the output.
4. Use 'Copy', 'Download', 'Save' or 'Load' buttons as needed.

Features:
- Real-world accuracy enforcement
- Prompt chaining logic
- Output customization (tone, format, audience)
- Save/load input state with localStorage
- Optional dark mode toggle

No internet connection required once loaded locally.
